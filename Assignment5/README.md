# Electronic Device Shopping Cart

## Description
OOP-based shopping cart system using inheritance.

## Features
- Device base class
- Smartphone, Laptop, Tablet subclasses
- Cart system
- Stock management
- Discounts
- Unit tests

## How to Run
python main.py

## Run Tests
python -m unittest tests/test_store.py
