# User Management System

## Description
This project demonstrates Object-Oriented Programming concepts:
- Instance Variables
- Class Attributes
- Class Methods
- Static Methods
- Unit Testing

## Classes

### User
Represents a user with personal information and age calculation.

### UserService
Manages users using class-level dictionary storage.

### UserUtil
Provides utility methods for:
- Generating user IDs
- Generating strong passwords
- Email validation

## How to Run

1. Clone repository:
git clone <repo-url>

2. Run program:
python main.py

3. Run tests:
python -m unittest tests/test_user.py
